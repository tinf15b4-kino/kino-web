### 1. Get information

#### 1.1 Brief Description

The get information function will allow every user to find informations about cinemas and movies.

A Movie have the following informations:
  - Ratings
  - Description 
  - Playtimes
  
A Cinema have the following informations:
  - Ratings
  - Adress
  - Playlist

No postconditions.
